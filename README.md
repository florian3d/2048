# 2048
2048 game clone in Python. My own Python challenge inspired by Daniel Shiffman from Coding Train. Written in PYTHON 3.4 using PYGAME library.
<br/><br/>
W:  UP<br/>
S:  DOWN<br/>
A:  LEFT<br/>
D:  RIGHT<br/>
R:  UNDO<br/>
N:  RESTART<br/>
H:  HELP<br/>
X:  EXIT<br/>
